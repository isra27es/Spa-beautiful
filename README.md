Beauty Spa - App Móvil para Clientes

Beauty Spa es una aplicación móvil diseñada para facilitar a los clientes el proceso de agendar citas en un SPA. La interfaz es intuitiva, moderna y enfocada en brindar una experiencia cómoda al usuario.

 Características principales

- Registro y login de usuarios con email, Google o Apple.
- Autenticación segura con opción de "Recuérdame por 30 días".
- Visualización de información del SPA (teléfono, correo, ubicación y redes sociales).
- Acceso a un perfil profesional con datos de contacto.
- Opción para contactar directamente vía WhatsApp.

 Rol del Usuario

Este proyecto está enfocado exclusivamente al **rol de cliente**, permitiendo:

- Crear cuenta nueva.
- Iniciar sesión.
- Navegar información institucional.
- Obtener medios de contacto y ubicación.

 Tecnologías utilizadas

- **Frontend móvil:** Android Studio con Jetpack Compose / Flutter / React Native (especificar según corresponda).
- **Backend/API:** Laravel (si aplica).
- **Autenticación:** JWT, Google Sign-In, Apple Sign-In.

 Capturas de Pantalla

- Pantalla de Login

 Manual de Usuario

El manual se encuentra disponible en formato Word. Contiene:

- Instrucciones detalladas por pantalla.
- Preguntas frecuentes.
- Guía para agendar citas paso a paso.

 Seguridad

La app sigue buenas prácticas de seguridad para proteger la información del cliente:
- Validación de credenciales.
- Sesiones protegidas.
- Consentimiento de términos y condiciones en el registro.
